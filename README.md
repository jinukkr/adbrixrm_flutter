# adbrixrm_flutter

AdBrixRM Plugin for Flutter Project
![Pub Version](https://img.shields.io/pub/v/adbrixrm_flutter)

## Support Platform

- Android 4.1 +
- iOS 10 +

## Current AdBrixRM SDK version

- Android :  2.0.0.8
- iOS : 1.6.5407

## Install the Plugin

Add dependencies on pubspec.yaml

```
dependencies:
  adbrixrm_flutter: ^1.3.0
```
